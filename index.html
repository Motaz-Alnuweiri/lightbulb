    Welcome to Lightbulb

    Lightbulb is an instant messenger for Symbian based on MeegIM and licensed on GPL 3 license. It is written using Symbian C++, Qt and QtQuick. Also uses qxmpp library for XMPP support.

    Features

    • sending, receiving messages
    • lots of customization options
    • notifications (info banner in app, discreet popup, notification led, vibration, and sound)
    • partial multi account support (you can add multiple ones, but only one can be enabled)
    • roster management
    • VCard, avatars and stuff
    • typing notifications
    • ... and more! (I love how this one sounds :D)

    Why would I need this anyway?

    Lightbulb IM can be used for Google Talk/Hangouts, Facebook Chat and is compatible with probably any XMPP server. It's beautiful and should handle your text messages correctly. Wanna use it for something else than XMPP? You should familiarize yourself with transports. I use it for Gadu-Gadu myself. :)
